# Hospital Appointment Management System
![Description of Image](images/pic1.png)

This is a **Hospital Appointment Management System** built with **HTML, CSS, PHP, and JavaScript**. It allows users to manage appointments efficiently with full **CRUD operations** (Create, Read, Update, Delete).

## Features
- **User Authentication:** Signup and Login functionality.  
- **Appointment Management:** Users can book, view, update, and delete appointments.  
- **Search Functionality:** Search appointments by patient name.  
- **Validation & Alerts:** Form validations with alerts and redirects using JavaScript.  
- **Database:** Uses **XAMPP (MySQL)** for storing and managing appointment data.  

## Technologies Used
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** PHP  
- **Database:** MySQL (via XAMPP)  

## How to Run
1. Install **XAMPP** and start Apache & MySQL.  
2. Import the provided database `.sql` file into **phpMyAdmin**.  
3. Place the project folder in `htdocs` directory of XAMPP.  
4. Open your browser and go to `http://localhost/<project-folder-name>`  

This project demonstrates the full **CRUD workflow** and basic user authentication, making it easy to manage hospital appointments.
